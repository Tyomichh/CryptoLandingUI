# CryptoLandingUI

Static multi-page UI prototype for a crypto-related landing website.
This project focuses on layout, styling, and UI structure.

## About the project

This project is a front-end UI demonstration only.
It does not include any backend logic, real authentication,
or payment processing.

All forms and interactive elements are visual previews created
for learning and practice purposes.

## Features

- Multi-page static layout (Home, Pricing, Sign Up, Payment Preview)
- Responsive design
- Pricing section with subscription plans (UI only)
- Sign up form (no data submission)
- Payment preview form with demo fields

## Payment preview notice

The payment page represents a UI preview only.

- No real payment data is collected
- No card or crypto transactions are performed
- All fields are demo placeholders for visual representation

## Tech stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Project status

Completed as a learning UI project.
